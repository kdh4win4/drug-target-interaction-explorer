# 💊 DTI Explorer — Bayesian Drug-Target Interaction Analysis

An AI-powered drug-target interaction (DTI) analysis tool built with Claude AI and Bayesian statistical modeling.

## 🚀 Live Demo
👉 **[Launch App](https://kdh4win4.github.io/dti-explorer/dti_explorer.html)**

## 🧬 Features
- **Bayesian posterior scoring** — Prior → Likelihood → Posterior update visualization
- **Binding affinity estimation** with 95% confidence intervals
- **MM-GBSA ΔG estimate** (kcal/mol)
- **Drug-likeness (QED)**, selectivity index, toxicity flag
- **Key binding residues** and pocket description
- **Similar approved drugs** comparison
- Zero installation — runs entirely in the browser

## 🔑 Requirements
You need a free Anthropic API key to use this app.
1. Go to [console.anthropic.com](https://console.anthropic.com)
2. Create an account → API Keys → Create Key
3. Paste your key into the app's API Key field

> Your API key is never stored or sent anywhere except directly to Anthropic's servers.
